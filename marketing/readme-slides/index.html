05   How It Gets There Five-Phase Process
  1. 01
    Look Backward
    Find prior handoff docs, project structure docs, the current task definition.
  2. 02
    Look at Present
    git diff · git log, decisions made, current state, what was deliberately not done.
  3. 03
    Look Forward
    Identify next task(s), check for ripple effects from this session, flag stale docs.
  4. 04
    Reading List
    Smallest set of files the next agent needs — ranked, absolute paths, stop-early friendly.
  5. 05
    Write & Commit
    Adapted handoff doc. Update STATUS.md. Snapshot commit. Output copy-paste prompt.
Deterministic about process — not output. No. 05 — Process
